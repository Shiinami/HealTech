Deskripsi

HealTech adalah rangkaian kegiatan kompetisi yang diselenggarakan oleh HIMA Informatika dan Program Studi Informatika ITS Kesehatan dr. Soepraoen Malang. Tujuannya adalah untuk meningkatkan kemampuan dan kreativitas generasi muda dalam teknologi â€” khususnya di sektor kesehatan.


Website ini berfungsi sebagai portal utama untuk:

Informasi acara: beranda, info, kompetisi, tentang, kontak. 

Pengumuman hasil, download guidebook. 

Formulaire / upload berkas peserta. 

Fitur Utama

Halaman pengumuman dan tahapan kompetisi. 

Detail masing-kompetisi (UI/UX Design Competition, dll) beserta pendaftaran dan timeline. 

Informasi kontak panitia: WhatsApp, Instagram. 

Rangkaian kompetisi (contoh: UI/UX Design Competition) dengan tim, biaya pendaftaran, tanggal.

Instalasi & Pengembangan

Clone repository:

git clone https://github.com/username/healtech2025-website.git


Instalasi dependencies jika ada (untuk framework/backend):

npm install      # atau composer install


Konfigurasi environment:

Buat file .env atau config.php sesuai kebutuhan

Atur koneksi database, profil admin, dsb

Jalankan aplikasi secara lokal:

npm run dev      # atau php artisan serve


Untuk deploy:

Pastikan konfigurasi produksi sudah benar

Lakukan build jika diperlukan (npm run build)

Upload ke server/hosting
